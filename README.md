# baffler
A transpiled language written in typescript that transpiles in vanilla js

## How to Baffle
> ### Types of Statements
> return: `return 123;`  
> function definitions: `fun main() [ disp(123); ]`  
> if/else statements:   `if 1 == 2 [ disp(1, 2); ] else [ disp(2, 1); ]`  
> constant assignment:  `set name = "Max";`  
> variable assignment:  `var age = 69; # nice`
> > #### Assignable Values
> > function calls: `var function = main();`  
> > arrays: `var ages = [17, 54, 15, 67];`  
> > regular values: `var age = 98; set name = "mark"; set id_1 = id_2;`  
> > expressions: `set an_expression = 1 + 2 * 3 / 4;`
